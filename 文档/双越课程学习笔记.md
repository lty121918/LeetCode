# 双越课程学习笔记

## 3.1

unshift, shift,splice时间复杂度是O（n）,其他的不是。

arr.slice(-2);截取最后两个位置的数组。

```js
console.log([1,2,3,4].slice(-2));//[3,4]
```

检测运行时间的方法：

```js
console.time("time");
xxxx执行的代码;
console.timeEnd("time");
```

## 3.2

实现栈：stack用push、pop方法

实现队列：queue用push、shift方法

set、object无序结构

数组、map、链表是有序结构

链表查询慢O（n），新增和删除快O（1）

数组查询快O（1），新增和删除慢O（n）
